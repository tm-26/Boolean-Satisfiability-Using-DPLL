In this project the Davis-Putnam-Logemann-Loveland (DPLL) algorithm is used to determine whether the entered Conjunctive Normal Form (CNF) Boolean expression is satisfiable.

booleanSatisfiability.py takes in one argument, this being an expresion. A list of example expressions can be found in CNF examples.txt. If no argument is given the user will be asked to 
enter an expression.

To run this project python3 is required
